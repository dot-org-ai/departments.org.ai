# Construction Operations

> Business-as-Code definition for the Construction Operations department category.

## Overview

Construction-specific functions including estimating, field operations, and project controls

## Departments

| Department | Code | Description |
|-----------|------|-------------|
| Estimating | EST | Cost estimation, quantity takeoff, bid preparation, and proposal development |
| Field Operations | FLDOP | On-site construction management, crew supervision, daily progress tracking, and field coordination |
| Project Controls | PCTRL | Cost tracking, schedule management, earned value analysis, and change order administration |
| Construction Safety | CSAFE | Job site safety inspections, toolbox talks, incident investigation, and OSHA compliance |
| Subcontractor Management | SUBM | Subcontractor prequalification, contract administration, performance evaluation, and lien waiver tracking |
| Building Information Modeling | BIM | BIM coordination, clash detection, 3D model management, and digital construction documentation |
