# Retail Operations

> Business-as-Code definition for the Retail Operations department category.

## Overview

Retail-specific functions including merchandising, store operations, and loss prevention

## Departments

| Department | Code | Description |
|-----------|------|-------------|
| Merchandising | MERCH | Assortment planning, buying, pricing strategy, markdown optimization, and vendor negotiation |
| Store Operations | STORE | Store management, staffing, cash handling, customer service standards, and daily operations |
| Loss Prevention | LP | Shrinkage reduction, theft investigation, exception-based reporting, and asset protection programs |
| Visual Merchandising | VMERCH | Store layout design, window displays, planogram management, and in-store brand presentation |
| Ecommerce | ECOM | Online storefront management, digital merchandising, order fulfillment, and omnichannel integration |
